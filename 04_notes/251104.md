# 🔊 Raspberry Pi 4 블루투스 스피커 연결 튜토리얼


## 🧩 1. 패키지 업데이트 및 설치

블루투스 관련 유틸리티와 오디오 툴을 설치

```bash
sudo apt install -y pulseaudio pulseaudio-module-bluetooth bluez alsa-utils
```
PulseAudio가 실행 중인지 확인
```
pactl info

# 정상
Server Name: pulseaudio
```

## 🔵 2. 블루투스 스피커 검색 및 페어링

```bash
# 1. 블루투스 설정 및 진입
bluetoothctl

# 2. 컨트롤러 활성화
power on
agent on
default-agent

# 3. 주변 장치 검색
scan on

# MAC 주소 확인 후 scan off
# (예: 12:11:2E:C7:40:DD 장치 이름)

# 4. 페어링 시도
pair 12:11:2E:C7:40:DD

# 5. 연결
connect 12:11:2E:C7:40:DD

# 6. 자동 연결을 위해 신뢰 설정
trust 12:11:2E:C7:40:DD

# 7. 종료
exit
```

## 🎧 3. 연결 확인

현재 연결된 오디오 장치를 확인

```bash
pactl list short sinks

```
예시 출력:
```
0   alsa_output.platform-bcm2835_audio.stereo-fallback   module-alsa-card.c  s16le 2ch 44100Hz   SUSPENDED
1   bluez_sink.12_11_2E_C7_40_DD.a2dp_sink               module-bluez5-device.c s16le 2ch 44100Hz SUSPENDED
```

## 🔁 4. 블루투스 스피커를 기본 출력으로 설정
```bash
pactl set-default-sink bluez_sink.12_11_2E_C7_40_DD.a2dp_sink
```
>⚠️ 12_11_2E_C7_40_DD 부분은 실제 스피커의 MAC 주소에 맞게 변경

## 🔊 5. 사운드 테스트

```
speaker-test -t wav -c 2
```
좌/우 채널에서 "Front Left", "Front Right" 소리가 나면 성공🎉



## ⚙️ 6. 부팅 시 자동 연결
라즈베리파이 부팅 시 자동으로 블루투스 스피커 연결
```
crontab -e

# 맨 아래에 추가 :
@reboot sleep 10 && echo -e "connect 12:11:2E:C7:40:DD\nquit" | bluetoothctl

```
