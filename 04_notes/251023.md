
## 라즈베리파이4 + 터치스크린 gui 실행 가이드

### 1. 하드웨어 연결
- 라즈베리파이4 + 터치스크린 (HDMI + USB/I2C)
- 케이블 → 영상 출력
- USB 케이블 → 터치 신호 (입력)
- 전원 공급이 충분해야 합니다. (터치스크린은 전류 소모 많음)

### 2. 장치 인식 확인
```
lsusb
ls /dev/input/
sudo evtest
```

### 3. GUI 실행 환경 설치

- 방법1. 최소한의 X 서버 환경
```
sudo apt update
sudo apt install xorg openbox
```

- 방법2. X 없이 바로 framebuffer에 출력
  - X를 설치하지 않고 `/dev/fb0` 에 직접 그리는 라이브러리 사용
   → Kivy, Pygame, PyQt(EGLFS) 등이 지원

### 4. Python GUI 설치 및 실행
예: Tkinter, PyQt5, Kivy 등 중 하나 선택

### 5. GUI 동작 구조
1. Python GUI 실행
2. GUI가 X서버 또는 프레임버퍼로 출력됨
3. 터치 입력이 `/dev/input/event*` → 드라이버 → GUI 프레임워크로 전달
4. Python 이벤트 루프에서 터치/클릭 이벤트 처리
5. (옵션) ROS2 노드와 통신: 버튼 누르면 토픽 publish / service call


### ⚙️ 선택적 구성 예시 (ROS2 포함)
```
[ROS2 Node: navigation_control.py]
       ↑
       │  (ros2 topic/pub/sub)
       ↓
[Python GUI: control_panel.py]  ← 터치스크린 입력
       ↑
       │  (touch input)
       ↓
[사용자: 손가락으로 터치]

```
